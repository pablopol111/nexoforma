# NexoForma Platform Starter

Base limpia del proyecto para:
- administrador
- nutricionista
- cliente
- Next.js App Router
- Supabase

## Qué incluye
- estructura inicial de carpetas
- cliente y servidor de Supabase
- middleware base
- páginas iniciales para admin, nutricionista y cliente
- esquema SQL inicial

## Orden recomendado
1. Crear proyecto en Supabase
2. Copiar `.env.example` a `.env.local`
3. Ejecutar `supabase/schema.sql`
4. Instalar dependencias
5. Arrancar en local

## Comandos
```bash
npm install
npm run dev
```

## Notas
- Esta es una base nueva y limpia
- Todavía no incluye autenticación completa ni paneles conectados a datos reales
- Está preparada para seguir construyéndola archivo por archivo
